Braço Robotico

Este projeto apresenta um protótipo de um braço robótico programado para executar movimentos precisos, utilizando servomotores controlados por um microcontrolador Arduino. O sistema foi desenvolvido com foco em aplicações educacionais e automação básica, demonstrando conceitos de eletrônica, mecânica e lógica de programação. A estrutura do braço foi projetada para simular tarefas como pegar e mover objetos, e o código permite controlar seus movimentos com eficiência, de forma autônoma. Feito com Arduino e C++, é um projeto que alia criatividade, aprendizado e inovação.

Estufa Autônoma

Este modelo 3D foi desenvolvido no Blender, utilizando elementos prontos para otimizar o tempo de criação, e apresenta uma estufa autônoma voltada para o cultivo inteligente de plantações. A animação tem o objetivo de ilustrar visualmente a ideia do projeto, que integra o braço robótico já implementado para realizar tarefas de plantio, manutenção e colheita, garantindo um ambiente controlado e eficiente.

O projeto foi apresentado na Feira Regional do Conhecimento, em Soledade - RS, como uma ideia inovadora para automação agrícola. Embora o braço robótico tenha sido implementado, as demais tecnologias como visão computacional, sensores ambientais e algoritmos inteligentes — não foram aplicadas na prática, sendo parte de uma proposta para um sistema 100% autônomo em um possível desenvolvimento futuro.

Essa ideia demonstra o potencial da integração entre robótica, visão computacional e automação para aumentar a produtividade e sustentabilidade na agricultura.
